Corrige el siguiente error de tipado en el proyecto Next.js (TypeScript):

Type error: Type '{ data: PageData; isMobile: false; }' is not assignable to type 'IntrinsicAttributes & { data: PageData; }'.
Property 'isMobile' does not exist on type 'IntrinsicAttributes & { data: PageData; }'.

Detalles del proyecto:

Framework: Next.js 14.2.33

Lenguaje: TypeScript

Archivo con error: /src/components/HomePageClient.tsx

Línea del error: 36

Instrucciones para Gemini

Abre el archivo /src/components/HomePageClient.tsx.

Busca el componente DesktopLayout que se renderiza con isMobile:

return <DesktopLayout data={data} isMobile={isMobile} />;

Revisa la definición de DesktopLayout (probablemente en /src/components/DesktopLayout.tsx).

Si el componente realmente utiliza el estado móvil:

Agrega la prop isMobile?: boolean; al tipo de props de DesktopLayout.

Define un valor por defecto false si no está presente.

Si isMobile no se usa dentro de DesktopLayout, elimina esa prop de la llamada en HomePageClient.tsx.

Verifica que el proyecto compile correctamente (npm run build).

Limpia importaciones innecesarias y optimiza el código.
